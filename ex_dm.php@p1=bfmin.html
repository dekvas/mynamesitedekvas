Минимальная ДНФ булевой функции

На этой странице вы найдете готовые примеры по булевой алгебре, связанные с минимизацией нормальных формул булевой функции. Помимо минимальной ДНФ, в процессе решения могут быть найдены ядровые импликанты и ядро функции.

Основные методы получения минимальной ДНФ: равносильные преобразования, метод карт Карно, метод Квайна-МакКласки. Все они разбираются в следующих задачах.

Задачи и решения о минимизации ДНФ булевых функций

Задача 1. Применяя равносильные преобразования привести булеву функцию $f=(\bar x \to \bar y)\to (yz \to \bar xz)$ к минимальной ДНФ.

Решение задачи о минимальной ДНФ

Задача 2. Для заданной логической функции: $$F= \overline {(\bar A \vee B\cdot \bar C)} \cdot \overline{( \overline {(B \downarrow C)}\cdot D)}$$
- найти дизъюнктивную нормальную форму;
- составить таблицу истинности и построить диаграмму Карно;
- получить минимальную дизъюнктивную нормальную форму;
- от минимальной ДНФ перейти к конъюнктивной нормальной форме.

Получение СДНФ и СКНФ

Задача 3. Для функции $f(x_1,x_2,x_3,x_4)$, заданной списком номеров наборов из $Nf$, методом Квайна найти сокращенную и минимальную ДНФ. Список номеров: 0,1,2,3,6,7,8,9,11,15.

Решение методом Квайна

Задача 4. С помощью карт Карно найти сокращенную, все тупиковые и минимальную ДНФ булевой функции $f(x_1,x_2,x_3,x_4)$, заданной вектором значений (1100 0101 0011 0011).

Решение методом карт Карно

Задача 5. Найти минимальные КНФ булевых функций от аргументов $A,B,C,D$. В квадратных скобках — неопределённые состояния:
$$f=(1,2,5,6,14),\ [4,9,11,12,15].$$

Решение о минимальной КНФ

Задача 6. Найти минимальные ДНФ и КНФ булевой функции от аргументов $A,B,C,D$:
$$f=(1,2,3,4,6,7,8,9,11,12,13,14,15).$$

Решение о нахождении минимальных КНФ и ДНФ

Задача 7. Для булевой функции $f(x,y,z)=(\bar x \vee \bar y)\wedge(\bar y \vee \bar z)\to(\bar x \vee \bar z)$ найти методом преобразований минимальную ДНФ. По таблице истинности построить СКНФ, по минимальной ДНФ — релейно-контактную схему.

Минимизация и построение РКС

Задача 8. Переключательная функция от трёх аргументов задана десятичным номером. Получить её таблицу истинности, СДНФ, СКНФ, минимизировать по кубу соседних чисел и карте Карно, реализовать минимальную схему на базисе И, ИЛИ, НЕ.

Минимизация переключательной функции

Задача 9. Для булевой функции $f$, заданной в таблице:
а) найти сокращённую ДНФ;
б) найти ядро функции;
в) получить все тупиковые ДНФ и указать минимальные;
г) на карте Карно отметить ядро и покрытия минимальных ДНФ.

Решение задачи о минимизации функции

Задача 10. Двумя способами (Кван и Карно) найти сокращённую, ядровую и все минимальные ДНФ для функции $f$, заданной вектором 0101101001001110. Построить минимальную функциональную и контактную схемы.
