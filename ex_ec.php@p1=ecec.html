Задачи по эконометрике с решениями

Задача 1. По группе предприятий, производящих однородную продукцию, известно, как зависит себестоимость единицы продукции $y$ от факторов, приведенных в таблице. Определите с помощью коэффициентов эластичности силу влияния каждого фактора на результат. Проранжируйте факторы по силе влияния, сделайте вывод.

Решение задачи по эконометрике 1

Задача 2. В таблице указаны парные коэффициенты корреляции. Проведите анализ целесообразности включения заданных факторов в уравнение множественной линейной регрессии.

Решение задачи по эконометрике 2

Задача 3. По некоторым территориям известны значения среднего душевого дохода $X$ и процент расходов на продовольственные товары $Y$. Рассчитайте параметры линейной, степенной и экспоненциальной моделей, выберите оптимальную по среднему отклонению и критерию Фишера.

Решение задачи по эконометрике 3

Задача 4. С использованием фиктивной переменной $D$ (есть ли дети) постройте модель $Y=b_0+b_2X_2+bD$, проверьте значимость коэффициентов и сделайте выводы.

Решение задачи по эконометрике 4

Задача 5. Постройте линии регрессии $Y$ на $X$ и $X$ на $Y$ для двумерной выборки, рассчитайте коэффициенты корреляции и детерминации.

Решение задачи по линейной регрессии

Задача 6. Постройте поле корреляции, найдите параметры парной линейной регрессии, оцените её достоверность, постройте доверительные интервалы и рассчитайте коэффициент эластичности.

Решение задачи 6

Задача 7. По данным о прибыли $Y$ и доле товара $X$ в грузообороте найдите параметры линейного уравнения, коэффициенты корреляции и детерминации, постройте график и сделайте вывод о качестве модели.

Решение задачи 7

Задача 8. В уравнении $y_i=20+700/x$ доля остаточной дисперсии равна 0,19. Найдите корреляционный индекс и проверьте значимость уравнения критерием Фишера ($\alpha=0,05$).

Решение задачи 8

Задача 9. По множественной регрессии $y_i=-1.725+0.903x_1+0.081x_2$ и средним значениям найдите эластичности факторов, ранжируйте их и рассчитайте скорректированный $R^2$.

Решение задачи 9

Задача 10. При тесте Голдфельда–Квандта ($S_1=0{,}07$, $S_2=0{,}92$, $k_1=k_2=6$, $\alpha=0{,}01$) проверьте гетероскедастичность регрессии $y=bx+a$.

Решение задачи 10

Задача 11. Применив условие идентификации, определите, идентифицированы ли уравнения данной модели.
