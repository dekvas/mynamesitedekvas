Примеры решений задач на C/C++

Задача 1. Односвязный список. Реализуйте список строк. Напишите функции для подсчёта количества слов в списке:
а. начинающихся и оканчивающихся одной и той же буквой
б. начинающихся с той же буквы, что и следующее слово
в. совпадающих с последним словом.

Исходные файлы проекта

Задача 2. Двумерный массив. Характеристикой столбца целочисленной матрицы назовём сумму модулей его отрицательных нечётных элементов. Переставляя столбцы заданной матрицы, расположить их в порядке возрастания характеристик.

Исходные файлы проекта

Задача 3. Калькулятор. Реализовать на C++ калькулятор с четырьмя арифметическими действиями. Запрашивать аргументы и операцию, «зациклить» программу (предлагать повторить вычисление) и выход по нажатию «q». Обработать деление на ноль и некорректный ввод операции.

Исходный код

Задача 4. Работа со строками. На входе массив строк формата «Фамилия Имя Отчество». Функция должна возвращать новый массив строк вида «И.О. Фамилия».
